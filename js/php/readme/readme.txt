デクリメントドリンクID

	書式:
	decrementDrink(ドリンクID)

	使い方例:
	decrementDrink(1);

	説明:
	指定のドリンクの在庫をマイナス１します。
	在庫がない場合は処理を行いません。

// ------------------------------------------------------------

ゲットセリフByID

	書式:
	getSerifById(セリフID)

	使い方例:
	$serif = getSerifById(1);
	print($serif);

	説明:
	指定したIDのセリフ内容を返します
